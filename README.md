# AI-powered-cybersecurity-threat-detection
system designed to detect and classify network threats such as brute-force attacks, port scans, and anomalous traffic in real-time. The system leverages data preprocessing, feature engineering, and a Random Forest classifier to analyze network traffic patterns and identify potential threats with high accuracy.

## 🚀 Overview

This project is an AI-powered cybersecurity system that detects and classifies network threats using machine learning.
It analyzes network traffic data and identifies:
* 🟢 Normal Traffic
* 🔴 Brute Force Attacks
* 🟠 Port Scanning
* 🟣 Anomalies
The system is designed with a modular architecture, making it scalable for real-time cybersecurity applications.
